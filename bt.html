Fontos tudnivalók a Web Bluetooth API-ról

  1. Hogyan működik a böngésző Bluetooth támogatása?

    A Web Bluetooth API nem listázza a párosított eszközöket. Ehelyett egy böngésző ablakban jeleníti meg az elérhető eszközöket, ahol manuálisan kell kiválasztani az eszközt.

  2. Lépések az eszköz csatlakoztatásához:
    1. Kattints a "Bluetooth eszközök keresése" gombra
    2. A böngésző megjelenít egy párbeszédablakot az elérhető eszközökkel
    3. Válaszd ki a kívánt eszközt (lehet, hogy először párosítani kell)
    4. A böngésző megkérdezi, hogy engedélyezed-e a kapcsolódást
  3. Korlátozások:
    • Csak biztonságos kontextusban működik (HTTPS vagy localhost)
    • Felhasználói interakció szükséges (nem indulhat automatikusan)
    • Nem minden Bluetooth profil támogatott
  4. Támogatott böngészők: Chrome, Edge, Opera (teljes), Firefox, Safari (korlátozott)
Fontos: A Bluetooth GPS adatok olvasásához az eszköznek támogatnia kell a megfelelő GATT szolgáltatásokat. A szimulált mód segít tesztelni az alkalmazást, ha nincs kompatibilis GPS eszközöd.
